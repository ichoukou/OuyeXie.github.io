# Terminal Short cut

1、将光标移动到行首：ctrl + a

2、将光标移动到行尾：ctrl + e

3、清除屏幕：            ctrl + l

4、搜索以前使用命令：ctrl + r

5、清除当前行：         ctrl + u

6、清除至当前行尾：   ctrl + k

7、单词为单位移动：option + 方向键

# Projection:
